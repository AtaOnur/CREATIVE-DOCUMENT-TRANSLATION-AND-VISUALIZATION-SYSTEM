================================================================================
  BITIRME PROJESI — TUM DOSYALAR, ISLEVLER VE MODIFIKASYON SORULARI (COZUMLU)
  Proje: Creative Document Translation and Visualization System (pdf_bitirme)
  Stack: ASP.NET Core 8 MVC + SQLite + EF Core + PDF.js + OCR + Multi-AI + Gemini TTS
================================================================================

NOT: wwwroot/lib/ altindaki Bootstrap/jQuery gibi ucuncu taraf kutuphaneler ve web/
Next.js iskeleti ayrica belirtilmistir. Asil sunum uygulamasi repo kokundeki MVC'dir.

Proje adi: Creative Document Translation and Visualization System
Repo adi: pdf_bitirme

Problem: PDF'teki belirli bir bolgeden metin cikarip AI ile cevirmek, ozetlemek,
         yeniden yazmak veya gorsellestirmek.

Cozum: PDF yukle → calisma alaninda bolge sec → OCR → metin uzerinde AI →
       sonuclari deftere kaydet → (opsiyonel) OCR metnini seslendir.

Teknik ozet: ASP.NET MVC + SQLite + EF Core + PDF.js + Tesseract/PaddleOCR
             (Windows) + coklu AI saglayici (Gemini, Groq, HuggingFace,
             Stability, Wolfram) + Gemini TTS.

Kapsam disi (bilincli): Rastgele fotograftan OCR yok; sadece PDF ici secili bolge.

22 — MUHTEMEL JURI SORULARI + KISA CEVAPLAR
================================================================================

1. Neden MVC?
   Katmanlar net, bitirme MVP'si icin yeterli ve savunulabilir.

2. OCR neden bolge bazli?
   Akademik PDF'lerde tam sayfa gurultu; kullanici ilgili paragrafi secer.

3. Mock vs gercek OCR?
   Ocr:Provider ile secilir; sunumda Mock, Windows'ta Tesseract/Paddle.

4. AI nasil coklu saglayici?
   MultiProviderAiService model ID → provider eslemesi (appsettings Ai:Models[]).

5. Migration yok mu?
   MVP'de EnsureCreated + manuel SQL; uretimde migration onerilir.

6. Identity neden yok?
   Zaman kisiti; arayuz ISimpleAccountStore ile degistirilebilir tasarlandi.

7. Seslendirme OCR motoruna bagli mi?
   Hayir; textarea'daki metin Gemini TTS ile okunur.

8. Admin paneli var mi?
   Evet — AdminController, rol tabanli [Authorize(Roles = "Admin")].
